# Pantallas

## Mínimas para MVP
1. **Login**
2. **Dashboard** (resumen de pendientes, alertas stock, entregas próximas)
3. **Clientes** (ABM básico)
4. **Solicitudes** (alta y detalle con adjuntos)
5. **Presupuestos** (editor + estados + export PDF)
6. **Insumos/Stock** (catálogo + movimientos)
7. **Agenda** (calendario + detalle de trabajo)
8. **Reportes simples** (consumo y estado de trabajos)

## Criterios UI
- Navegación lateral por módulos.
- Acciones primarias destacadas.
- Estados con colores y etiquetas consistentes.
